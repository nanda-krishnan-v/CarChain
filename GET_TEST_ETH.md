# How to Get Test ETH for Free

## Why Do You Need Test ETH?

Even on **test networks**, you still need ETH to pay for **gas fees**. Gas fees are required for:

- Deploying smart contracts
- Calling functions that modify blockchain state (like `registerCar`)
- Any transaction that writes to the blockchain

**Good news:** Test ETH is **completely free** from faucets! You just need to request it.

## Step-by-Step Guide

### Step 1: Check Your Current Network

1. Open MetaMask
2. Look at the network dropdown (top center)
3. Note which test network you're on:
   - **Sepolia** (Chain ID: 11155111)
   - **Holesky** (Chain ID: 17000) - formerly Hoodi
   - **Goerli** (Chain ID: 5)

### Step 2: Get Your Wallet Address

1. In MetaMask, click on your account name/address at the top
2. Click the copy icon to copy your wallet address
3. It will look like: `0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb`

### Step 3: Use a Faucet

Choose a faucet based on your network:

#### For Sepolia Test Network:

1. **[sepoliafaucet.com](https://sepoliafaucet.com/)**

   - Paste your wallet address
   - Complete CAPTCHA
   - Click "Send Me ETH"
   - Wait 1-5 minutes

2. **[infura.io/faucet/sepolia](https://infura.io/faucet/sepolia)**

   - Requires Twitter/X account
   - Paste your wallet address
   - Tweet the required message
   - Receive 0.5 ETH

3. **[faucet.quicknode.com/ethereum/sepolia](https://faucet.quicknode.com/ethereum/sepolia)**
   - Paste your wallet address
   - Complete verification
   - Receive test ETH

#### For Holesky Test Network (Hoodi):

1. **[holesky-faucet.pk910.de](https://holesky-faucet.pk910.de/)**

   - Paste your wallet address
   - Complete CAPTCHA
   - Click "Request"
   - Wait for confirmation

2. **[faucet.quicknode.com/ethereum/holesky](https://faucet.quicknode.com/ethereum/holesky)**
   - Paste your wallet address
   - Complete verification
   - Receive test ETH

#### For Goerli Test Network:

1. **[goerlifaucet.com](https://goerlifaucet.com/)**

   - Paste your wallet address
   - Complete CAPTCHA
   - Receive test ETH

2. **[faucets.chain.link/goerli](https://faucets.chain.link/goerli)**
   - Requires GitHub account
   - Paste your wallet address
   - Complete verification
   - Receive 0.1 ETH

### Step 4: Wait for Confirmation

- Test ETH usually arrives within 1-10 minutes
- Check your MetaMask balance
- You should see the test ETH appear in your wallet

### Step 5: Verify You Have Enough

- **Minimum needed:** Usually 0.001 ETH is enough for several transactions
- **Typical faucet amount:** 0.1 - 0.5 ETH (plenty for testing!)
- **Gas cost per transaction:** Usually 0.0001 - 0.001 ETH

## Troubleshooting

### "Insufficient funds" Error

If you see this error in MetaMask:

1. **Check your balance:** Open MetaMask and see how much test ETH you have
2. **Get more test ETH:** Use a faucet (see above)
3. **Wait for confirmation:** Sometimes transactions take a few minutes
4. **Check network:** Make sure you're on the correct test network

### Faucet Not Working

- **Try a different faucet:** Some faucets have daily limits
- **Wait 24 hours:** Many faucets have cooldown periods
- **Complete all requirements:** Some faucets require social media verification
- **Check network:** Make sure you're requesting ETH for the correct network

### Still Having Issues?

1. **Verify network:** Make sure MetaMask shows the correct test network
2. **Check address:** Double-check you copied the correct wallet address
3. **Try multiple faucets:** Different faucets have different requirements
4. **Wait longer:** Some faucets can take up to 30 minutes
